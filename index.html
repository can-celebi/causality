What's Covered in Session 1

This session covers core R programming fundamentals:

  • Working with data types (numeric, character, logical, factors)
  • Operators, logical statements, and indexing
  • Control flow (for-loops and conditional statements)
  • Writing and using functions
  • Working with dates (parsing and manipulation)
  • Creating and manipulating data frames
  • Importing and exporting data (CSV, TSV, Excel, SPSS, Stata)
  • Data manipulation with Base R and Tidyverse
  • Basic visualization and statistical analysis
