About These Sessions

This repository contains materials for a two-part R programming and causal inference course. Session 1 is essential and covers R fundamentals necessary for data analysis. Session 1.5 is optional and provides conceptual grounding in causal inference through directed acyclic graphs (DAGs).

Each session includes lecture slides, problem sets, textbook references, and R code templates with multiple dataset variations.
